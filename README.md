# Sesión Interactiva: Hiperplasia Benigna de Próstata

Esta es una aplicación web interactiva diseñada para seguir el caso clínico de José Rodriguez en tiempo real.

## Roles y Acceso

### Estudiantes
- **Ingreso**: Únicamente con correo institucional.
- **Flujo**: Verán la descripción del caso y un formulario para enviar sus percepciones. La vista se actualiza automáticamente cuando el docente avanza la sesión.

### Docente
- **Ingreso**: Correo `ricardomd2000@gmail.com` y su clave de Firebase.
- **Control**: Tablero para avanzar los "4 Momentos" del caso y ver las respuestas de los alumnos en vivo.

## Momentos del Caso
1. **Clínico**: Percepciones iniciales sobre el paciente.
2. **Anatomía/Histología**: Correlación con los síntomas.
3. **Fisiopatología**: Mecanismos de la HBP y farmacología.
4. **Manejo**: Recomendaciones finales y cierre.

## Instrucciones para el Docente
1. Ejecuta `npm run dev` para iniciar localmente.
2. Ingresa como **Docente**.
3. Usa los botones **Anterior** y **Siguiente** para guiar a los estudiantes.
4. Observa cómo aparecen las respuestas en la columna derecha de tu monitor.
